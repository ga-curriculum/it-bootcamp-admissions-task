## Goal

The goal of the admissions assessment for the Cybersecurity Bootcamp is to create a Google Doc and share it with your admissions advisor so they are able to view it. You can practice this assessment by following the steps below.

## Steps

1. Sign into your Google account or create one if you don't already have one. If you're not sure how to do this or need help, please refer to the Google Account Help [documentation for creating a Google account](https://support.google.com/accounts/answer/27441).

2. Create a new Google Doc. See the Google Docs Editors Help [documentation to create, view, or download a file](https://support.google.com/docs/answer/49114) if you're unsure how to do this.

3. Rename the document, giving it a title of your choice. Refer to the documentation provided in the previous step, and follow the steps in the **Rename a file** section if you need help.

4. Make the document publicly available. If you're unsure how to do this, check out the Google Docs Editors Help [documentation on making Google files public](https://support.google.com/docs/answer/183965).

5. After completing the steps above, anyone should be able to view your document. You can check this by copying the URL to your document and pasting it into a new *private* browser window. If you can see the document, then it is publicly available.
